# java-coding-challenges
Este projeto é uma coleção de desafios de lógica em Java, acessados por um menu interativo de console. Ótimo para:  Treinar algoritmos  Reforçar fundamentos de Java  Criar portfólio  Se preparar para entrevistas técnicas  Cada desafio foi implementado em sua própria classe, seguindo uma interface comum para manter o projeto organizado e escalável.
